üîê Authentication Endpoints

POST/api/auth/register/ - Register new user

GET/api/auth/profile/ - Get user profile

üí∞ Transaction Endpoints

GET/api/categories/ - List all categories
POST/api/categories/ - Create new category
GET/api/transactions/ - List user transactions (with filtering)
POST/api/transactions/ - Create new transaction
GET/api/transactions/{id}/ - Get transaction details
PUT/api/transactions/{id}/ - Update transaction
DELETE/api/transactions/{id}/ - Delete transaction

üìä Reports & Export

GET/api/monthly-summaries/ - Get monthly summaries
GET/api/dashboard/ - Get dashboard statistics
GET/api/export-csv/ - Export transactions as CSV

üîç Filtering & Search Parameters

Transaction filtering supports:

  • start_date & end_date - Date range filtering
  • category - Filter by category ID
  • category_type - Filter by 'income' or 'expense'
  • amount_min & amount_max - Amount range filtering
  • search - Search in description and category name
  • ordering - Sort by date, amount, created_at (add '-' for descending)

Example: /api/transactions/?category_type=expense&start_date=2024-01-01&search=food
